## Usage
We can connect to AniList the same way we connect to MyAnimeList here.
```
GET https://geniusanime.com/widgets/myanimelist?username=[MAL_USERNAME]
```

### Themes
```json
["default", "tokyonight", "rule34", "dracula", "nord", "catppuccin", "gruvbox", "monokai"]
```
